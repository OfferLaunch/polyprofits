# Assets

Brand and media assets for PolyProfits.

## Structure

- **images/logos/** – `logo-primary.png` (nav), `logo-white.png` (footer)
- **images/emblems/** – `emblem-blue.png`, `emblem-black.png` (favicons)
- **images/placeholders/** – SVG placeholders (favicon, og-image, photo)

Replace placeholders with final assets when ready.
